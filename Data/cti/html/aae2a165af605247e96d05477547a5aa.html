Introduction to the GNU Accounting Utilities

The GNU Accounting Utilities provide login and process accounting utilities for GNU/Linux and other systems.

Downloading GNU acct

GNU acct can be found on http://ftp.gnu.org/gnu/acct/ [via http] and ftp://ftp.gnu.org/gnu/acct/ [via FTP]. It can also be found on one of our FTP mirrors.

Documentation

GNU acct documentation can be found at http://www.gnu.org/software/acct/manual/accounting.html. You may also be able to read the manual by typing info accounting at the shell prompt on your system.

Mailing Lists/Newsgroups

GNU acct currently does not have a mailing list. Announcements about GNU acct are made on the project page on Savannah. Support requests, feature requests and bug reports may also be filed there.

Request an Enhancement

If you would like a new feature to be included in future versions of GNU acct, use the tech support manager on Savannah to file a feature request.

Please remember that development of GNU acct is a volunteer effort, and you can also contribute to its development. For information about contributing to the GNU Project, please read How to help GNU.

Report a Bug

If you think you have found a bug in GNU acct, then you should file as complete a bug report as possible in the bug tracking system on Savannah. If you would like to submit a patch, please use diff -u5 to generate the patch, and give a detailed description.

Maintainer

The GNU Accounting Utilities are currently being maintained by Markus Gothe <nihilus [at] member.fsf.org>. The project is managed online at Savannah. Current development is done by Ciaran O'Riordan, Manuel A. Fernandez Montecelo, Tim Schmielau and Markus Gothe.
